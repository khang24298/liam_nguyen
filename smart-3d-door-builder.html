Smart 3D Door Builder

Client: SOS Design

Working with SOS Design we were asked to pitch to create a new website and door configurator Smart’s designer door range. They had an existing door configurator that dynamically rendered 2D door designs using images and svgs. It was functionally OK, and quite clever behind the scenes but visually it was looking tired and was not user friendly.

Smart have a range of 50+ doors that are highly customisable and they were looking for a ‘best in class’ way of showing them off and demonstrating how bespoke they are.

We experimented with a few ideas making some scrappy prototypes and eventually settled on using 3D. The existing print material user realistic 3D renders so we already had a decent amount of 3D assets to work with and it showcased the door in the best light.

Using Three.js I built the configurator with the ability to change colours, hardware and glazing options. This all tied into an API that sent the door designs off to installers in the local area. The site is full responsive and performant on mobile as well as desktop. We used draco compression on the 3D models to keep the filesize nice and small.

We tied it all together with a WordPress site showing off all the doors features and connected the two with favouring colours and styles within the website. This became short cuts that you could use a starting off point to jump into the configurator.

It was a really fun project to be involved in, and I enjoyed being part of the creative process and helping to shape the ideas.
